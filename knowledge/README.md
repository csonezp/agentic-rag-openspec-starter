# 本地 Markdown 知识库

这个目录是项目的第一个知识源，也是后续 RAG 学习的小型测试夹具。

Phase 2 实现基础 RAG 时，默认从这里读取 `*.md` 文件，完成文档加载、切片、embedding、检索和引用回答。

## 初始约定

- 文件格式优先使用 Markdown。
- 每个文件应有清晰标题。
- 后续切片时保留文件路径、标题层级和段落位置。
- 这里先放学习用资料，不放真实敏感数据。
- 评测问题不放在本目录，避免后续被索引后污染检索评测；评测问题放在 `evals/`。

## 当前内容设计

本目录的文档故意保持小而有区分度：

- 有些文档回答明确事实，方便验证“可回答问题”。
- 有些文档主题相近但边界不同，方便验证检索是否误召回。
- 有些限制条件会用于后续验证拒答和安全边界。
