# 项目协作约定

## 文档语言

- 本项目的说明文档、进度文档、学习笔记、运行手册、任务记录等面向人的文本内容统一使用中文。
- 代码标识符、包名、命令、环境变量名、第三方 API 名称保持英文，不强行翻译。
- 后续新增或更新 `README.md`、`docs/` 下的文档、session log、todo/task 文档时，默认使用中文。
