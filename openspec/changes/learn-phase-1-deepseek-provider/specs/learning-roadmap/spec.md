## ADDED Requirements

### Requirement: Phase 1 DeepSeek real provider learning

Phase 1 MUST 包含一个 DeepSeek 官方 API 真实模型对接学习小节，用于验证当前 Agent 抽象可以适配用户实际使用的模型服务商。

#### Scenario: 推进 DeepSeek provider 小节

- **WHEN** 用户要求在 Phase 1 中接入 DeepSeek 官方 API
- **THEN** 路线图 MUST 增加对应小节
- **AND** 小节完成标准 MUST 包含至少一次真实 DeepSeek 模型调用验证
